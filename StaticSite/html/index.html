Застосунки для бізнесу

A2v10 це сучасна програмна платформа для розробки швидких, компактних та зручних бізнес застосунків будь якої складности

Про платформу
Відкритість

Сирцевий код системи повністю доступний для перегляду і вивчення. Всі прикладні застосунки фактично є набром пласких файлів, які дуже зручно підтримувати в будь-якій системі спільної розробки програмного забезпечення.

Оптимізовано для хмари

Код платформи оптимизовано спеціально для роботи в хмарному середовищі. Мінімізована кількість запитів до сервера застосунків, що дає високу швидкість роботи. Також мінімізована кількість запитів до бази даних. Це значно зменьшує витрати на володіння системою.

Мультітенантність

Платформа підртимує мультітенантість. Це означає, що можлива пітримка багатьох незалежних сегментів баз даних в одній фізичній базі данних. Мультітенантні застосунки ідеально підходять для постачання програмного забезпечення "як сервіс" (SaaS).

Ефективність

Aрхітектура платформи побудована таким чином, щоб розробник бізнес-застосунків міг зосередитися на вирішенні прикладних задач предметної області, максимально абстрагуючись від низькорівневих технологій.

Web + Desktop

Підтримується два виконуючих середовища. Ваш прикладний застоcунок буде працювати без жодних змін в web-оточенні, в хмарному середовищі і на настільному комп'ютері.

JavaScript

Всі бізнес-застосунки розробляються з використанням мови програмування JavaScript, що є однією з найбільш популярних в світі. Більш складні задачі можна вирішувати за допомогою будь якої мови .NET Framework.
