Завантажити

На цій сторінці зібрано все, що потрібно для розробки бізнес-застосунків на платформі A2v10. NuGet пакети для розробників, інсталяційні пакети, додаткові модулі та розширення.

NuGet пакети для NET.Framework

  • A2v10.Web.Runtime. Повна версія платформи. Підтримується NET framework 4.6.1.
  • A2v10.Web.Mvc. Пакет для розробки сайтів з використанням платформи в режимі Site-mode.

NuGet пакети для NET.Core

  • A2v10.Data.Interfaces. Абстракції для роботи з моделями даних платформи в застосунках .NET.Core.
  • A2v10.Data.Core. Основний пакет для роботи з моделями даних платформи в застосунках .NET.Core
  • A2v10.Data.Providers. Пакет для роботи з зовнішніми джерелами даних (*.dbf, *.csv, etc).

Пакети інсталяції (msi)

  • A2v10.Web.Runtime. Інсталяційний пакет повної версії платформи.

Тестування

  • A2v10.Tests.Browser. Інсталяційний пакет модуля тестування застосунків А2v10 безпосередньо в браузері.

Робота з зовнішнім обладнанням

  • A2v10.Browser.Companion. Інсталяційний пакет Native Messaging Host для роботи з зовнішнім обладнаннями через розширення Chrome.
  • A2v10 Equipment Host. Розширення Chrome. Працює спільно з Native Messaging Host. Призначене для для роботи з зовнішнім обладнанням безпосередньо з браузера.

Попередні версії А2

Утіліти (сирцевий код)

  • A2.Replication. Система реплікації баз даних для Microsoft SQL-Server.
  • DW.Loader. Data Warehouse для Microsoft SQL-Server.
