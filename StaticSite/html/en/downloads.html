Downloads

On this page, you will find everything you need for the development of the A2v10 business applications. These are the installation packages, NuGet packages for developers, Chrome extensions, etc.

NuGet packages (NET.Framework)

  • A2v10.Web.Runtime. Full A2v10 platform. (NET framework 4.6.1)
  • A2v10.Web.Mvc. Package for Site-mode development.

NuGet packages (NET.Core)

  • A2v10.Data.Interfaces. A2v10 Data Model abstractions.
  • A2v10.Data.Core. A2v10 Data model. Full package.
  • A2v10.Data.Providers. A2v10 Data Model. Workinng with external data providers (*.dbf, *.csv, etc).

Installation packages (msi)

  • A2v10.Web.Runtime. A2v10 full version installation package.

Testing

  • A2v10.Tests.Browser. Browser test module installation package (Chromium).

Working with external equipment

  • A2v10.Browser.Companion. A2v10 Native Messaging Host installation package. Designed to works with external equipment directly from the browser.
  • A2v10 Equipment Host. Chrome Extension. Communicate with A2v10 native messaging host.

A2. Previous versions

Utilities (source code)

  • A2.Replication. Lightweight simple replication system for Microsoft SQL-Server.
  • DW.Loader. Simple Data Warehouse for Microsoft SQL-Server.
